Chủ đề 4: Ứng dụng Quản lý Thư viện Sách
  • Mô tả: Ứng dụng quản lý sách trong một thư viện nhỏ.
  • Các thực thể (MySQL): Sach (Mã sách, Tên sách, Tác giả, Năm xuất bản, Thể loại, Số
    lượng), TheLoai (Mã TL, Tên TL).
  • Chức năng chính:
    o Thêm, Sửa, Xóa, Xem danh sách sách.
    o Tìm kiếm sách theo tên, tác giả hoặc thể loại.
    o Chức năng mượn/trả sách đơn giản (cập nhật số lượng).
