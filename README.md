Chủ đề 4: Ứng dụng Quản lý Thư viện Sách<br>
  • Mô tả: Ứng dụng quản lý sách trong một thư viện nhỏ.<br>
  • Các thực thể (MySQL): Sach (Mã sách, Tên sách, Tác giả, Năm xuất bản, Thể loại, Số
    lượng), TheLoai (Mã TL, Tên TL).<br>
  • Chức năng chính:<br>
    o Thêm, Sửa, Xóa, Xem danh sách sách.<br>
    o Tìm kiếm sách theo tên, tác giả hoặc thể loại.<br>
    o Chức năng mượn/trả sách đơn giản (cập nhật số lượng).<br>
